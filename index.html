HTML Basics

HTML (HyperText Markup Language) is the standard markup language for creating web pages.

Key HTML elements include:

  • <h1> to <h6> - Headings
  • <p> - Paragraphs
  • <a> - Links
  • <img> - Images
  • <div> - Divisions

CSS Basics

CSS (Cascading Style Sheets) is used to style and layout web pages.

CSS can control:

  • Colors
  • Fonts
  • Spacing
  • Layouts
  • Responsive design

JavaScript Basics

JavaScript makes web pages interactive by responding to user actions.

Basic JavaScript includes:

  • Variables
  • Functions
  • Event Listeners
  • DOM Manipulation
